# Rubino Social Media Playbook

A comprehensive social media strategy playbook for Rubino Shoes, created in partnership with Haven Creative Studio.

## Deploying to GitHub Pages

### Option 1: Quick Deploy (Recommended)

1. Create a new repository on GitHub (e.g., `rubino-playbook`)
2. Upload `index.html` to the repository
3. Go to **Settings** â†’ **Pages**
4. Under "Source", select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at: `https://[your-username].github.io/rubino-playbook/`

### Option 2: Using Git Command Line

```bash
git init
git add .
git commit -m "Initial playbook deployment"
git branch -M main
git remote add origin https://github.com/[your-username]/rubino-playbook.git
git push -u origin main
```

Then enable GitHub Pages in repository settings.
